Design Principles
👆
Touch-First
Minimum 48×48px tap targets. Thumb-reachable primary actions. No hover-only affordances.
⚡
Perceived Speed
Skeleton screens, optimistic UI, sub-200ms micro-interactions. Never show a spinner where a skeleton works.
💎
Luxury Tone
Every animation, spacing, and shadow must feel premium. No generic admin aesthetics. Gold on green.
🌐
RTL-Safe
Every mobile pattern uses logical CSS properties (inset-inline, padding-inline). Never directional hacks.
♿
Accessible
Focus rings on keyboard, ARIA labels, reduced-motion variants for every animation.
📐
Content-First
Chrome collapses on mobile. Nav hides. Headers compact. Content gets the screen.
